从零开始完成登录页和个人中心页的实践，完整地走了一遍 Android 客户端开发流程：从创建项目、写 XML 布局、用findViewById 绑定控件，到给按钮和条目加点击事件、用 Toast 做反馈，再到用 SQLite 预埋账号密码做真实登录验证，并用 SharedPreferences 保存和读取用户名、签名，还学会了导入图标资源和自己控制图标大小与排版。中间也踩了不少坑，比如大括号位置、XML 注释写错位置、资源命名不规范等，但在不断改错、重新运行的过程中，对 Activity 与布局的关系、数据存储方式以及报错定位方式都有了更直观和系统的理解。
